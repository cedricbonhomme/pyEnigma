pyEnigma
========

#### Python Enigma cypher machine simulator.

Presentation
------------
Python 2 -> Python 3

Usage
-----

    $ echo "Hello World" | ./cypher_enigma_safe.py secret B  I II V 
    Vjpfw Ekdxj

    $ echo "Vjpfw Ekdxj" | ./cypher_enigma_safe.py secret B  I II V
    Hello World
